# Photoshop на React (GrayBit‑7)

Веб-приложение для работы с изображениями в форматах **PNG**, **JPG** и **GB7** – учебном 7‑битном формате оттенков серого с возможностью маски. Реализован полный цикл: загрузка, отображение на canvas, экспорт.

## 🖼️ Ссылка на хостинг

Сайт доступен на Render:  
[https://photoshop-ndbeliaeb0401.onrender.com](https://photoshop-ndbeliaeb0401.onrender.com)

## 🚀 Функциональность

- Загрузка изображений (перетаскиванием или через кнопку) – PNG, JPG, GB7.
- Отрисовка загруженного изображения на HTML5 canvas с автоподгоном под размер окна.
- Статус‑бар: ширина, высота, глубина цвета (24 бита для PNG/JPG, 7/8 бит для GB7).
- Экспорт текущего canvas в PNG, JPG или GB7.
- Адаптивный интерфейс:
  - На десктопе – вертикальный тулбар слева.
  - На мобильных устройствах – горизонтальный тулбар с выпадающим меню экспорта.
- Собственная реализация декодера и кодера **GB7** (без внешних библиотек).

## 🧠 Формат GB7

GrayBit‑7 – учебный формат без сжатия, где каждый пиксель хранится в 1 байте:
- **7 младших бит** – значение оттенка серого (0–127)
- **Старший бит** – бит маски (прозрачность / выделение)

Структура файла: сигнатура `GB7\x1D`, версия, флаги, ширина/высота (big-endian), резерв, затем массив пикселей.  
Кодер и декодер написаны вручную с использованием `ArrayBuffer` и `DataView` – полное соответствие спецификации лабораторной работы.
