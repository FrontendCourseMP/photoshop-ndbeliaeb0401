## 📌 Описание проекта

Веб-приложение для работы с изображениями – реализация учебного проекта, включающая:

- Поддержку форматов **PNG, JPG** и собственного формата **GB7** (7‑битный grayscale с возможной маской).
- Полный цикл: загрузка, отображение на canvas, экспорт.
- Работу с цветовыми каналами (RGB, RGBA, Grayscale, Grayscale+Alpha).
- Инструмент **«Пипетка»** – получение цвета пикселя с координатами X,Y, RGB и CIELAB.
- Градационную коррекцию **«Уровни»** (Black, Gamma, White) с гистограммой и live preview.
- **Масштабирование** (12%–300%) с выбором интерполяции (билинейная / ближайший сосед).
- **Изменение размера** изображения (в процентах или пикселях) с сохранением пропорций и тултипом.
- **Фильтрацию через свёртку** (ядро 3×3) с предустановками, выбором канала и стратегией обработки краёв. Асинхронный Web Worker для тяжёлых вычислений.

## 🚀 Функциональность

### 🔹 Основные операции (лаб. 1)
- Загрузка PNG, JPG, GB7 (drag‑and‑drop или клик).
- Отображение на canvas с адаптацией под размер окна.
- Экспорт в PNG, JPG, GB7.
- Статус‑бар: ширина, высота, глубина цвета (32/24 бита для PNG/JPG, 7/8 бит для GB7).

### 🔹 Цветовые каналы и пипетка (лаб. 2)
- Панель каналов с миниатюрами и переключением режимов:  
  `RGB`, `RGBA`, `Grayscale`, `Grayscale+Alpha`.
- Инструмент «Пипетка» – клик по canvas (с учётом масштаба) показывает X,Y, RGB и CIELAB.

### 🔹 Градационная коррекция «Уровни» (лаб. 3)
- Модальное окно с гистограммой (линейный/логарифмический режим).
- Настройка чёрной точки, гаммы, белой точки.
- Выбор канала: Master, Red, Green, Blue, Alpha.
- Live Preview (в реальном времени), сброс, отмена, применение.

### 🔹 Масштабирование и изменение размера (лаб. 4)
- Ползунок масштаба (12–300%) в панели информации.
- Выбор метода интерполяции (билинейная / ближайший сосед).
- Модальное окно изменения размера: выбор единиц (проценты / пиксели), сохранение пропорций, тултип с описанием методов, валидация ввода.

### 🔹 Фильтрация через ядро (лаб. 5)
- Модальное окно с ядром 3×3, editable.
- Предустановки: identity, sharpen, gaussianBlur, boxBlur, prewittX, prewittY.
- Применение к выбранному каналу (все, R, G, B, A).
- Обработка краёв: копирование, чёрный, белый.
- Web Worker – асинхронная свёртка без блокировки UI, live preview.

## 🧪 Технологии

- **React 19 + Vite**
- **CSS Modules** (адаптивный дизайн)
- **react-dropzone** (drag & drop)
- **react-icons** (иконки)
- Web Workers (асинхронная фильтрация)
- Собственная реализация:
  - кодека GB7 (чтение/запись)
  - интерполяции (bilinear, nearest neighbour)
  - гистограммы, уровней, свёртки
